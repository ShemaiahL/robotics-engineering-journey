# Day 1 - What is Arduino?
Date: May 28, 2026

## Overview
Arduino is a tool used for controlling electronics. Arduino has three aspects to it, the hardware (arduino board), Arduino IDE, Arduino code.

## What I Learned
I learned that Arduino is like a small easy to use computer brain used to interact software programs to (hardware) electronic gagets. For example it can connect sensors, lightsm and motors.
ALso learned that inputs are electronic devices that gather information and outputs are electronic devices that do things.Arduino reads inputs and control outputs.THe microcontroller is what 
enables someone to read the inputs and control outputs. So Arduino is a physical hardware that uses an microcontroller, theres the Arduino development Environment called the ArduinoIDE and
the Arduino code itself called a sketch and that code is loaded onto the Arduino board.

## Key Terms
- **Microcontroller** - small programmable computer
- **Arduino IDE** (Integrated Development Environment) -software application (where you wirte your code)
- **Arduino code** (aka: sketch) - the software that is loaded onto theArduino board (gets loaded on microcontroller)

## Examples
-  Input devices can be sensors
-  Output devices can be motors, speakers, LED lights.
